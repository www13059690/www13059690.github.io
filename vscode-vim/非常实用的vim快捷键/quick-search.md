# 非常使用的vim技巧速查
## Normal模式
### 跳转类
```
gd          // Go to definition,跳转到定义。
ctrl + o    // 跳转后退回，比如如果gd到其他文件中去之后，可以用这个命令跳回来
space       // 进入简易模式（需开启），后可以快速跳转
gh          // 等于把鼠标移到光标在的这个单词上面，可以快速查看定义和报错。
w           // 向后移动一个单词
b           // 往前移动一个单词
gg          // 到文件头部
G           // 到文件结尾
NG          // N是一个数字表示行数，跳转到某一行
```
### 操作文本类
```
x           // 删除一个字符
dw          // 删除一个单词
cw          // 删除一个单词并进入insert模式
```
### 翻屏类
```
ctrl + f    // 下翻一屏
ctrl + d    // 下翻半屏
ctrl + b    // 上翻一屏
ctrl + u    // 上翻半屏
zz          // 将当前行移动至屏幕中央（不动光标，只动页面）
zt          // 将当前行移动到屏幕顶端
zb          // 将当前行移动到屏幕底端
```
### 插入类
```
o           // 在当前行的下面插入一行并切换到insert模式
O           // 在当前行的上面插入一行并切换到insert模式
a           // 在光标后插入
i           // 在光标前插入
```
## Visual模式
### 选择区域
```
v           // 字符选择
V           // 行选择
af          // 依据语法分析，将选择区域向外扩展
```
### 操作类
```
y           // 复制
p           // 粘贴
d           // 剪切
```
